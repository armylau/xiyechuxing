# TabBar 图标说明

请在当前目录添加以下图标文件：

## 图标列表

| 文件名 | 说明 | 尺寸 |
|--------|------|------|
| home.png | 首页默认图标 | 81x81px |
| home-active.png | 首页选中图标 | 81x81px |
| attraction.png | 景点默认图标 | 81x81px |
| attraction-active.png | 景点选中图标 | 81x81px |
| food.png | 吃住默认图标 | 81x81px |
| food-active.png | 吃住选中图标 | 81x81px |
| my.png | 我的默认图标 | 81x81px |
| my-active.png | 我的选中图标 | 81x81px |

## 图标规范

1. 格式：PNG
2. 尺寸：81 x 81 像素
3. 风格：建议线框风格，选中版本填充颜色
4. 颜色：默认图标用灰色(#999)，选中图标用主色(#4F6DF5)

## 获取图标

可以使用以下方式获取图标：

1. **Iconfont** - https://www.iconfont.cn 下载免费图标
2. **Feather Icons** - https://feathericons.com
3. **Heroicons** - https://heroicons.com

下载后将图标转换为 81x81px 的 PNG 格式即可使用。
